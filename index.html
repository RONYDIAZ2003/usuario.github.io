Webscraping

El webscraping es el proceso de extraer datos de sitios web mediante el uso de programas automatizados. Esta técnica es fundamental en la ciencia de datos ya que permite recolectar información valiosa que no está disponible en bases de datos estructuradas.

Contexto del problema: El webscraping es una técnica utilizada para extraer información de sitios web de manera automatizada. Este proceso es esencial en la ciencia de datos, ya que permite obtener datos valiosos que no están disponibles en bases de datos estructuradas. En este ejemplo, se aplicaron técnicas de webscraping para extraer información de dos sitios web distintos: Wikipedia y Books to Scrape.

Técnicas de Visualización

Contexto del problema: El análisis y la visualización de datos son fundamentales en la ciencia de datos, ya que permiten transformar datos complejos en representaciones visuales claras y comprensibles. En este ejemplo, utilizamos datos simulados de ventas mensuales de dos productos durante un año, así como la concentración de ventas por región, para aplicar diversas técnicas de visualización. Medidas de Tendencia Central

Las medidas de tendencia central son estadísticas que describen el centro de un conjunto de datos. Las más comunes son la media, la mediana y la moda. Estas medidas son esenciales para resumir grandes volúmenes de datos y proporcionar una visión general de la ubicación de los datos en una distribución.

Contexto del problema: Las medidas de tendencia central son estadísticas fundamentales en el análisis de datos, ya que nos permiten resumir un conjunto de datos con un solo valor representativo. En este ejemplo, utilizamos una lista de velocidades de autos en una carrera de Fórmula 1 para calcular la media, mediana y moda, proporcionando una visión clara de la distribución de estas velocidades.

Medidas de Dispersión

Las medidas de dispersión cuantifican la variabilidad de los datos alrededor de la tendencia central. Las más comunes son la varianza, la desviación estándar y el rango. Estas medidas son importantes para entender la consistencia y la variabilidad de los datos.

Contexto del problema: Las medidas de dispersión son estadísticas utilizadas para describir la variabilidad o dispersión de un conjunto de datos. En este ejemplo, trabajamos con una lista de velocidades de autos en una carrera de Fórmula 1 para analizar cómo se distribuyen estas velocidades. A través de una tabla de frecuencias y un histograma, podemos visualizar la consistencia y variabilidad de las velocidades registradas.
